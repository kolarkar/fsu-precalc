    Section 1.2 Piecewise Functions (ALEKS 1.7)

    Graph piecewise-defined functions.

    A piecewise function is a function built from pieces of different functions over different intervals.
    🔗
    🔗
    The absolute value function \(f\left( x\right) = \vert x \vert\) can be written as a piecewise function.
    \begin{equation*} f\left( x\right) =\left\{ \begin{array}{cc} x \amp \text{for }x\geq 0 \\ -x \amp \text{for }x\lt 0 \end{array} \right. \end{equation*}
    🔗

    Example 1.2.2.

    Graph the function.
    🔗
    \begin{equation*} g\left( x\right) =\left\{ \begin{array}{cc} -4 \amp \text{for }-4\leq x\lt -1 \\ -x \amp \text{for }-1\leq x\lt 1 \\ \sqrt{x-1} \amp \text{for }x\geq 1 \end{array} \right. \end{equation*}
    Solution.
    🔗
    🔗
    REMARK: When a function has no "gaps" it is said to be continuous. Informally, this means we can draw the function without lifting our pencil from the page. The formal definition of a continuous function will be studied in calculus.
    🔗

    Example 1.2.4.

    Suppose that the function \(f\) is defined for all real numbers as follows.
    🔗
    \begin{equation*} f\left( x\right) =\left\{ \begin{array}{cc} 6 \amp \text{if }x\lt -4 \\ -10+x^{2} \amp \text{for }-4\leq x\lt 4 \\ 2x-2 \amp \text{for }x\geq 4 \end{array} \right. \end{equation*}
    Graph the function \(f\text{.}\) Then determine whether or not the function is continuous.
    🔗
    Solution.
